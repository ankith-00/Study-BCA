EXPERIMENT - 02 : Study different types of network cables.


TRANSMISSION MEDIA
A communication channel that is used to carry the data from one transmitter to the receiver through the electromagnetic signals. The main function of this is to carry the data in the bits form through the Local Area Network(LAN). There are 2 types :
     – Guided Transmission media | wired
     – Unguided media | Wireless



WIRED
It is also referred to as Wired or Bounded transmission media. Signals being transmitted are directed and confined in a nerrow pathway by using physical links. There are two types :
    – Copper cable
    – Fiber optics

Features :
    – High Speed
    – Secure
    – Used for caparatively shorter distance



WIRELESS
It is also referred to as Wireless or Unbounded transmission media. No physical medium is required for the transmission of electromagnetic signals.

Features :
    – The signal is broadcasted through air
    – Less secure
    – Used for larger distance



COPPER CABLES
A copper cable is a group of two or more copper wires bundled together in a single sheath or jacket. Copper wire and cable is used in power generation, power transmission, power distribution, telecommunications, electronics circuitry, and countless types of electrical equipment. There are two types of copper cables :
    – Coaxial Cable
    – Twisted Pair Cables


  • COAXIAL CABLE

    A coaxial cable is an electrical cable with a copper conductor and an insulator shielding around it and a braided metal mesh that prevents signal interference and cross talk. Coaxial cable is also known as coax.
  • TWISTED PAIR CABLES

    One of the earliest guided transmission media is twisted pair cables. A twisted pair cable comprises of two separate insulated copper wires, which are twisted together and run in parallel. The copper wires are typically 1mm in diameter. One of the wires is used to transmit data and the other is the ground reference. There are two types of twised pair cables :
        – Unshielded Twisted Pair ( UTP )
        – Shielded Twisted Pair ( STP )
    • Unshielded Twisted Pair ( UTP )
      A twisted pair includes two insulated conductors twisted together in the spiral form, as shown in the figure. It can be shielded with a plastic cover. The UTP cables are very low-cost and simple to install.
    • Shielded Twisted Pair ( STP )
      In this, each insulated twisted pair are shielded by a metal foil or braided mesh. This mesh is also known as a metal shield. It decreases the interference of the disturbance caused by the surrounding. This property makes the cable bulky and expensive.

FIBER OPTICS



An Optical Fiber is a cylindrical fiber of glass which is hair thin size or any transparent dielectric medium. The fiber which is used for optical communication is waveguides made of transparent dielectrics. There are two types of fiber optic cables :
    – Single mode
    – Multi-mode

Required meterials :

    - RJ-45 connectors (at least two per cable)

  • T568A wire arrangement order :

    1.White/Green
    2. Green
    3. White/Orange
    4. Blue
    5. White/Blue
    6. Orange
    7. White/Brown

  • T568B wire arrangement order :

    1. White/Orange
    2. Orange
    3. White/Green
    4. Blue
    5. White/Blue
    6. Green
    7. White/Brown
    8. Brown

Straight-wire Cable Implementation :
Both ends use the same wiring standard (T568A-T568A or T568B-T568B).

Crossover Cable Implementation :
One end follows T568A, and the other end follows T568B. Wires can arranged in desire sequence based on wire being used.


STEP - 2 : Cut the cable to the required length. Strip the outer jacket (about 1-1.5 inches) using a cable stripper.


STEP - 3 : Insert Wires into the RJ-45 Connector. Hold the RJ-45 connector with the clip facing down. Insert the wires fully into the connector, ensuring each reaches the end.


STEP - 4 : Place the RJ-45 connector inside the crimping tool. Squeeze firmly to crimp the metal pins onto the wires. Repeat the process for the other end.


STEP - 5 : Verify the Connection. Use a cable tester to check if the cable is working correctly.





EXPERIMENT - 04 : Study the network IP address configuration ( Classification of Address, Static and Dynamic address ).

IP ADDRESS :
An Internet Protocol Address/IP Address is a unique identifier assigned to every device on TCP/IP network. The Internet Protocol is the set of rules that outlines how data should be transported across the Internet or local networks. IP Addresses help to identify devices and allow them to communicate with each other.
Classification of IP Address :
    – Static IP Address
    – Dynamic IP Adress



STATIC IP ADDRRESS
Static IP Address is the one that you configure yourself by editing your computer's network settings. In simple words it is an Address that does not change once your device is assigned to a static IP address, that number typically stays the same until the device is decommissioned or your network architecture changes.


DYNAMIC IP ADDRESS
Dynamic IP Address: It is a temporary address for devices connected to a network that continually change over a time. They're assigned by the dynamic host configuration(DHCP), a service running on the network. Dynamic IP address are issued using leasing system, meaning that the IP address is only active for limited time. If the lease expires, the computer will automatically request a new lease.

EXPERIMENT - 06 : Study of network devices ( Switch, Router and Brigde )

Network devices help in data communication and management in a network.

  • SWITCH

    A switch is a networking device that connects multiple devices within a LAN and forwards data only to the intended recipient.
    Example :
    Connecting computers, printers, and other devices in a local network.
    - Operates at the Data Link Layer of the OSI model.
    - Uses MAC addresses to forward data.
    - Supports multiple simultaneous data transmissions.
    - Reduces network congestion by providing dedicated bandwidth to devices.


  • ROUTER

    A router is a networking device that connects different networks and directs data between them.
    Example :
    Connecting a home or office network to the internet.
    - Operates at the Network Layer of the OSI model.
    - Uses IP addresses to forward data between networks.
    - Supports Network Address Translation (NAT) for internet access.
    - Can provide firewall and security features.


  • BRIDGE

    A bridge is a device that connects and filters traffic between two or more network segments.
    Example :
    Connecting two LAN segments while keeping them as a single network.
    - Operates at the Data Link Layer of the OSI model.
    - Uses MAC addresses to filter and forward data.
    - Reduces collision domains by dividing a large network into smaller segments.
    - Improves network performance by managing traffic.
